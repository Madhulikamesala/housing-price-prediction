# UK House Price Prediction — Machine Learning Pipeline

## Overview
An end-to-end machine learning pipeline to predict UK housing prices 
using advanced AI and deep learning techniques.
Built as part of MSc Computer Science — University of Hertfordshire (2025).

## Results
- 95% predictive accuracy
- Models compared: Random Forest, XGBoost, Deep Learning (Keras/TensorFlow)

## Technologies Used
- Python 3
- Pandas, NumPy
- Scikit-learn
- TensorFlow & Keras
- Matplotlib & Seaborn
- Google Colab

## Key Features
- Multi-source data integration
- Exploratory data analysis and visualisation
- Feature engineering and preprocessing
- Explainable AI (SHAP) for model interpretability
- Hyperparameter optimisation

## Dataset
UK Government Housing Price Dataset (publicly available)
